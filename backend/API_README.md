### text2img

```
description: generate image according to text prompt
para: prompt, negative prompt
res: link to the generated image
```

### img2img

```
description: generate new image according to text prompt and the original image
para: prompt, negative prompt, image_name, image(base64 format)
res: link to the generated image
```

### text2video

```
description: generate video according to text prompt
para: prompt, negative prompt, image_name, image(base64 format)
res: link to the generated video
```
